✨ Concept：思考を軽く、生きやすく
現代の忙しい日々の中で、つい止まってしまう思考。
このアプリは、言葉を介さずに「色」と「音」だけで、自分自身や大切な誰かとつながり、心を空っぽにするための場所です。

* **自分を許す時間**: 何もしない、考えない時間を肯定します。
* **色彩の共鳴**: 10秒かけてゆっくりと混ざり合うグラデーションが、張り詰めた心を解きほぐします。
* **想い**: 最後に静かな言葉。

##  Tech Stack
- **Frontend**: p5.js (Generative Art)
- **Deployment**: Vercel
- **Future Vision**: Antigravity, Dify を軸としたシンプルで心地よい体験の拡張
